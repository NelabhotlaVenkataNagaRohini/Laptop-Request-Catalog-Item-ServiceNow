# Laptop Request Catalog Item – ServiceNow Project

##  Project Overview
This project demonstrates the creation and implementation of a **Laptop Request Catalog Item** in ServiceNow.  

The system allows users to request a laptop through the Service Catalog with dynamic behavior using:
- Variables
- UI Policies
- UI Actions
- Update Sets
- Testing & Deployment

The project follows a structured SDLC approach divided into 7 phases as per SmartInternz guidelines.

---

##  Project Objectives
- Create a custom catalog item in ServiceNow
- Implement conditional UI behavior using UI Policies
- Add client-side functionality using UI Actions
- Export and import update sets between instances
- Perform functional and performance testing
- Document the complete development lifecycle

##  Key Functional Feature
When the user selects **"Additional Accessories"**,  
the **Accessories Details** field becomes:
- Visible
- Mandatory

This behavior is implemented using UI Policy & UI Policy Action.
